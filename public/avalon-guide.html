阿瓦隆玩法介绍

阿瓦隆是一款阵营推理桌游。玩家分为好人和坏人两个阵营，坏人隐藏身份混在好人中。 好人需要完成 3 次任务，坏人需要破坏 3 次任务。即使好人赢了任务，坏人还有一次机会刺杀关键角色来翻盘。

怎么赢？

  • 好人赢：成功完成 3 次任务
  • 坏人赢：破坏 3 次任务，或者在终局成功刺杀梅林

每回合做什么？

  • 1. 队长组队 → 提议出征队员
  • 2. 全员投票 → 同意或反对这个队伍
  • 3. 执行任务 → 队员秘密投成功或失败
  • 4. 公开结果 → 看任务是否成功

信息差异

  • 坏人：互相知道谁是同伴
  • 梅林：知道谁是坏人，但不能暴露
  • 派西维尔：知道谁"像梅林"，但分不清真假
  • 忠臣：什么都不知道，纯靠推理

角色一览

梅林 好人

知道谁是坏人。要暗中引导好人，但不能太明显，否则会被刺客盯上。

派西维尔 好人

知道哪两个人中有一个是梅林（另一个是莫甘娜）。要保护真梅林。

忠臣 好人

没有任何额外信息。靠观察发言、投票和任务结果来判断谁是坏人。

刺客 坏人

知道同伴是谁。任务失败要靠他，终局刺杀梅林也要靠他。

莫甘娜 坏人

在派西维尔眼里长得像梅林。用来迷惑派西维尔、保护真正的坏人。

爪牙 坏人

普通坏人。配合队友投票、发言，帮坏人阵营制造混乱。

不同人数的配置

人数越多，坏人越多，任务也越难。7 人及以上时，第 4 轮任务需要 2 张失败票才会失败。

人数 好人阵营 坏人阵营 各轮任务人数 第4轮失败票 备注
5 人 梅林 + 2 忠臣 刺客 + 莫甘娜 2 / 3 / 2 / 3 / 3 1 张 最紧凑，没有派西维尔
6 人 梅林 + 派西维尔 + 2 忠臣 刺客 + 莫甘娜 2 / 3 / 4 / 3 / 4 1 张 派西维尔加入，博弈更丰富
7 人 梅林 + 派西维尔 + 3 忠臣 刺客 + 莫甘娜 2 / 3 / 3 / 4 / 4 2 张 首次出现双失败规则
8 人 梅林 + 派西维尔 + 3 忠臣 刺客 + 莫甘娜 + 爪牙 3 / 4 / 4 / 5 / 5 2 张 坏人增加到 3 人
9 人 梅林 + 派西维尔 + 4 忠臣 刺客 + 莫甘娜 + 爪牙 3 / 4 / 4 / 5 / 5 2 张 公共信息更多，推理链更长
10 人 梅林 + 派西维尔 + 5 忠臣 刺客 + 莫甘娜 + 爪牙 3 / 4 / 4 / 5 / 5 2 张 最大场面，配合要求最高

好人怎么玩？

  • 梅林：暗中引导，不要直接点名坏人
  • 派西维尔：保护真梅林，注意莫甘娜的伪装
  • 忠臣：多看投票记录和任务结果，不要只听发言

坏人怎么玩？

  • 前期：表现得像好人，获得信任
  • 中期：争取加入关键任务队伍
  • 后期：记住梅林的线索，为刺杀做准备
